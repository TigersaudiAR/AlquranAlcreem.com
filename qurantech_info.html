– البرامج الحاسوبية والتطبيقات الذكية
– البرامج والتطبيقات التعليمية
– الأنظمة المرئية والصوتية التي لها علاقة بالقرآن الكريم
– أنظمة النشر المكتبي
– المواقع والتطبيقات الإلكترونية التعليمية

دليل الاستخدام

The database contains the following fields:

1- id (int): Auto_increment Number
2- page (int): Page Number
3- sura_name (varchar): Sora Name in Arabic (Uthman Taha Naskh font)
4- aya_no (int): Aya Number
5- quranic_words (text): Text of the Quranic word in (UthmanicHafs font)
6- explanation (text): Text of the Tafseer of the Quranic word in (Uthman Taha Naskh font)

It includes two folders:

– It contains folder of MuyassarGhareeb fonts (Uthmanic Hafs font and Uthman Taha Naskh font)
2- It contains files for developers (Excel, CSV, HTML5, SQL, XML and JSON).
